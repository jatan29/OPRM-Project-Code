EMSR-b (Expected Marginal Seat Revenue — version b): For each fare class j+1, aggregate all higher fare classes 1..j into a virtual class with weighted-average fare f̄1:j. Protect yj seats where P(D1:j ≤ yj) = (f̄1:j − fj+1) / f̄1:j. Booking limit for class j+1 = C − yj. Capacity C = 164 (A320 Neo).

EMSR-b Formula (Step-by-Step)
// Step 1: Aggregate demand & fare for classes 1..j λ_agg = Σ λᵢ (i = 1 to j) f̄_1:j = Σ(fᵢ × λᵢ) / Σ λᵢ // Step 2: Critical Ratio CR = (f̄_1:j - f_{j+1}) / f̄_1:j // Step 3: Protection level (Normal approx for Poisson) // D_1:j ~ Poisson(λ_agg) ≈ Normal(λ_agg, √λ_agg) y_j = λ_agg + Φ⁻¹(CR) × √(λ_agg) // Step 4: Booking limit B_{j+1} = C - y_j
Critical Ratio vs Fare Class
Insight: The critical ratio CR = (f̄ − fnext) / f̄ determines how aggressively we protect. With high CR (low fare gap), we protect fewer seats. With low CR (next class is much cheaper), we protect more aggressively. EMSR-b aggregation raises f̄ above individual class fares, justifying stronger protection.
Newsvendor Overbooking Model: With no-show rate p, sell C + y* seats where optimal y* satisfies the critical ratio P(N ≤ y*−1) = r / (r + b). Here N ~ Binomial(C, p) ≈ Normal(Cp, √Cp(1−p)), r = average fare collected, b = cost of denied boarding (refund + compensation). Capacity C = 164.

Key Recommendations
Implement EMSR-b: Switching from FCFS to EMSR-b protection levels yields an estimated +35.9% revenue improvement per flight (₹2.48L/flight). With ~1,400 annual Delhi-Mumbai flights, this is ₹34.7Cr/year.

Demand segmentation: Close Super Saver (B1) entirely. Tighten Saver (B2) to 24 seats. This alone accounts for 78% of the EMSR-b gain by protecting high-yield inventory in D-7 to D-1 window.
